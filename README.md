# Personal Website

This is my personal website hosted on GitHub Pages. It features a minimalist design and showcases my professional background, projects, and contact information.

## Features

- Clean, minimalist design
- Responsive layout
- Easy navigation
- Contact information
- Social media links

## Technologies Used

- HTML5
- CSS3
- GitHub Pages
- Jekyll

## Setup

1. Fork this repository
2. Clone your forked repository
3. Make your changes
4. Push to your repository
5. Enable GitHub Pages in your repository settings
